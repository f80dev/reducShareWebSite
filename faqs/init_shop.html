Créer votre point de vente


La création d'un point de vente précéde toujours la création d'une promotion, car toutes les promotions
sont rattachés aux points de vente

Le nom du point de vente peut être

  • le nom de l'enseigne, de l'établissement
  • Votre propre pseudo
Les réductions fabriquées sur ce point de vente peuvent être visible sur la carte des promotions. En revanche, cela n'est possible que si votre compte est authentifié (par mail, facebook ou google).

Les points de vente peuvent être taggés. Cela permet notamment aux clients de les retrouver plus facilement sur la carte des promotions.
