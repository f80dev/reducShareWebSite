Créer votre point de vente


Avant de pouvoir lancer une promotion, il faut toujours créer le point de vente auquel elle est ratachée. Celui-ci peut être physique avec une adresse postale, ou virtuel avec une adresse internet (en développement).
Le nom du point de vente peut être
  • le nom de l'enseigne, de l'établissement
  • Votre propre pseudo
Les réductions fabriquées sur ce point de vente peuvent être visible sur la carte des promotions. En revanche, cela n'est possible que si votre compte est authentifié (par mail, facebook ou google).

Les points de vente peuvent être taggés. Cela permet notamment aux clients de les retrouver plus facilement sur la carte des promotions.
Enfin vous pouvez changer l'icône de votre point de vente avec un visuel de votre choix qui peut être une photo ou une image extraite des banques de photos que REDUCSHARE utilise.

Une fois créé, votre point de vente peut être modifié. Depuis ce point de vente vous pouvez :
  • Lancer des promotions,
  • Editer une affiche permettant à vos clients de récupérer votre dernière promotion,
  • Analyser la diffusion de vos dernières promotions pour optimiser les suivantes
